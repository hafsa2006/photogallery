📸 Dream Gallery

Dream Gallery is a modern and responsive photo gallery website created using HTML and CSS. It provides an elegant interface to display beautiful images with smooth animations and a clean user experience.

Features
✨ Modern UI Design
📱 Fully Responsive Layout
🎨 Glassmorphism Navbar
🖼️ Responsive Image Gallery
💖 Smooth Hover Animations
🌈 Gradient Background
⚡ Fast Loading
🌙 Clean & Minimal Design
Technologies Used
HTML5
CSS3
Flexbox
CSS Grid
